# Inventaire - Après chapitre 2

## Lieux clés identifiés

### Villa des Mouettes
- Statut : Base d'opération des enfants
- Occupants : François, Mick, Claude, Annie, Dagobert, Oncle Henri, Tante Cécile, Maria
- Particularité : Vue sur le promontoire et le vieux phare depuis les chambres
- Accès : Chemin descendant vers le village de Kernach

### Village de Kernach
- Statut : Exploré par les enfants
- Type : Petit port de pêche avec maisons en granit gris
- Lieux spécifiques :
  - La boulangerie de M. Quéméner (façade blanche, enseigne en fer forgé)
  - Le port avec barques colorées (bleu cobalt, rouge vif, vert bouteille)
  - Les quais où Loïc répare ses filets et nasses
- Atmosphère : Odeur de sel, varech, poisson frais - mouettes criaillantes
- Habitants : Pêcheurs, M. Quéméner (boulanger), Loïc (vieux pêcheur), villageois divers, Yann et l'homme brutal

### Le vieux phare abandonné ("le phare du Diable")
- Statut : Non encore visité mais exploré à distance
- Localisation : Sur un promontoire rocheux qui devient une île à marée haute
- État : Abandonné depuis 1935, condamné, interdit d'accès
- Dangers connus : Escaliers vermoulus, planchers pourris, courants traîtres
- Contenu présumé : Lanterne brisée, rats, mouettes
- Légende : Gardien M. Le Goff disparu dans une tempête en 1935
- Particularité : Des signaux SOS y ont été vus la nuit précédente
- Accès : Possible uniquement à marée basse selon Loïc

### Le nouveau phare (en activité)
- Statut : Fonctionne normalement (mentionné chapitre 1)
- Particularité : Clignote régulièrement

## Connaissances acquises

### Historique du phare (1935)
- Gardien : M. Le Goff
- Incident : Disparition lors d'une terrible tempête avec "des vagues hautes comme des maisons"
- Résultat : Le gardien n'a jamais été revu
- Conséquence : Abandon total du phare depuis cette date

### Code Morse (François)
- Compétence : François connaît le code Morse grâce aux scouts
- Signal identifié : SOS = trois points, trois traits, trois points
