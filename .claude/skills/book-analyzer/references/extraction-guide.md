# Extraction Guide

Detailed checklists for book analysis.

## Style Extraction Checklist

### Narrative Voice
- [ ] POV type identified
- [ ] Narrator tone described (distant/intimate, serious/playful)
- [ ] Direct reader address noted (yes/no with examples)

### Tenses
- [ ] Primary tense identified with percentage
- [ ] Secondary tenses listed with usage context
- [ ] Tense shifts patterns noted

### Vocabulary
- [ ] 20-30 characteristic words/expressions collected
- [ ] 10-15 interjections listed
- [ ] Register identified (formal/casual/childlike/archaic)
- [ ] Era-specific terms noted
- [ ] Forbidden modern terms identified (by absence)

### Sentence Structure
- [ ] 50 sentences sampled
- [ ] Average length calculated (words)
- [ ] Min/max range noted
- [ ] Complexity pattern described

### Dialogue
- [ ] Quotation style identified (« » or " ")
- [ ] All dialogue tags listed with frequency
- [ ] Attribution position noted (before/after)
- [ ] Dialogue-to-prose ratio estimated

### Description
- [ ] Primary senses ranked
- [ ] Simile/metaphor usage noted
- [ ] Description length characterized
- [ ] Techniques identified (enumeration, comparison, etc.)

## Character Extraction Checklist

### Identity
- [ ] Full name extracted
- [ ] Nicknames collected
- [ ] Age determined (exact or approximate)
- [ ] Role classified (protagonist/supporting/antagonist)

### Appearance
- [ ] All physical descriptions collected with sources
- [ ] Distinctive features identified
- [ ] Clothing/style noted

### Personality
- [ ] 3-5 core traits identified
- [ ] Each trait supported by 2+ scenes
- [ ] Strengths listed with evidence
- [ ] Flaws listed with evidence
- [ ] Fears identified from reactions
- [ ] Desires inferred from goals

### Speech Patterns
- [ ] 5-10 representative quotes collected
- [ ] Verbal tics identified
- [ ] Favorite expressions listed
- [ ] Register noted (formal/casual)
- [ ] Tone characterized

### Relationships
- [ ] Each relationship mapped
- [ ] Dynamic described (protective, competitive, etc.)
- [ ] Evolution across story noted

### Behavioral Patterns
- [ ] Stress reactions documented
- [ ] Conflict behavior noted
- [ ] Joy expression described

---

## Structure Extraction Checklist

### Overall Arc
- [ ] Act breaks identified with chapter numbers
- [ ] Inciting incident located
- [ ] Midpoint identified
- [ ] Climax located
- [ ] Resolution summarized

### Chapter Analysis
- [ ] Total chapter count
- [ ] Word count range (min/avg/max)
- [ ] Opening patterns categorized
- [ ] Ending patterns categorized

### Recurring Patterns
- [ ] Meal/food scenes frequency
- [ ] Night/danger correlation
- [ ] Discovery patterns
- [ ] Adult involvement patterns
- [ ] Resolution patterns

### Mystery Mechanics (if applicable)
- [ ] Clue distribution mapped
- [ ] Red herrings identified
- [ ] Solution reveal method documented
- [ ] Reader vs character knowledge tracked

---

## Universe Extraction Checklist

### Location Identity
- [ ] Name and type classified
- [ ] Importance level assigned
- [ ] Story function described

### Physical Description
- [ ] All descriptive passages collected
- [ ] Layout/geography noted
- [ ] Key areas listed
- [ ] Hidden elements documented

### Atmosphere
- [ ] Default mood characterized
- [ ] Sights listed
- [ ] Sounds listed
- [ ] Smells listed
- [ ] Weather associations noted

### Time Variations
- [ ] Day appearance/mood
- [ ] Night appearance/mood
- [ ] Seasonal changes (if any)

---

## Quality Validation

Before finalizing output, verify:

- [ ] Every trait has quoted evidence
- [ ] Quantitative data includes sample sizes
- [ ] No invented details (only extracted)
- [ ] All quotes are verbatim
- [ ] Chapter references are accurate
- [ ] Templates are followed exactly
