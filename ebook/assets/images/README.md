# Images pour l'ebook

## Image de couverture (cover.jpg)

Placez votre image de couverture ici avec le nom `cover.jpg`.

| Propriete | Valeur recommandee |
|-----------|-------------------|
| Dimensions | 1600 x 2400 pixels |
| Ratio | 2:3 (portrait) |
| Format | JPEG ou PNG |
| Taille max | 5 MB |

## Images d'en-tete de chapitre (headers/)

Placez les images d'en-tete dans le sous-dossier `headers/`.

| Propriete | Valeur recommandee |
|-----------|-------------------|
| Dimensions | 800 x 300 pixels |
| Ratio | Large (paysage) |
| Format | JPEG |
| Taille max | 500 KB par image |

Nommez-les selon le chapitre : `chapitre-01.jpg`, `chapitre-02.jpg`, etc.
